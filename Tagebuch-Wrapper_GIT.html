      Herzlich Wilkommen

      Dies scheint das erste mal zu sein das Sie dieses Tradingtagebuch nutzen.
      Hier ein paar Hinweise dazu.

      Das Tagebuch speichert Daten nur lokal auf Ihrem Computer. Es werden also keine Daten über das Internet versendet.
      (Trotzdem sollten Sie während der Nutzung immer online sein, damit die volle Funktionalität gewährleistet ist.)

      Dadurch das alle Daten lokal gespeichert werden ergeben sich folgende Einschränkungen:

      • Sie können dieses Tagebuch nur auf DIESEM Computer mit DIESEM Browser nutzen.
      • Sollten sie den Browser oder den Computer wechseln, würden Sie ein neues Trdingtagebuch erstellen. (Das alte bleibt aber im Normalfall bestehen).
      • natürlich können Sie jederzeit ein Backup der Datenbank erstellen, falls Sie doch einmal den Browser oder Computer wechsel möchten.

      WICHTIG!
      Damit Ihre Daten beim beenden / schließen des Browsers nicht automatisch gelöscht werden, sollten Sie die automatische Löschung der "Cookies und Webseitendaten" abschalten. Wie Sie dies machen erfahren sie hier
      für Google Chrome,
      für Firefox und
      für Microsoft Edge.

      Bevor es los geht, werden aber erst noch ein paar Inputs benötigt:
      Bitte geben Sie nachfolgend Ihren Kontostand und den zugehörigen Brocker ein. Außerdem sollten Sie eine Auswahl treffen, ob für diesen Broker Steuern abeführt werden oder nicht.
      Werden keine Steuern abgeführt, wird von einem Geweinntrade keine Steuer abgezogen. D.h. Sie sehen ausschließlich netto Werte.


      Neue Zeile

      Geben Sie bitte hier die Höhe der Steuern die Sie abgeben müssen, sowie die Höhe des gemeldeten Freibetrages an. Außerdem können Sie wählen welche Steuern berechnet werden sollen.
      Sind Sie z.B. nicht Kirchensteuer pflichtig so deaktivieren Sie bitte hier das Häckchen.
      Dies bewirkt dass diese Steuer nicht in Ihr Ergebniss (GuV) einberechnet wird.

      Nichtsdestotrotz habe Sie die Möglichkeit ein Steuerformular für Ihren Steuerberate zu erstellen.
      Erstellen können Sie dieses in der 'Einstellungen' Seite.

      % Kapitalerttragssteuer
      % Solidaritätszuschlag
      % Kirchensteuer
      Freibetrag
